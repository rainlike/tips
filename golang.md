Golang
======

**helpful links**
* [index](https://golang.org/)
* [examples](https://github.com/adonovan/gopl.io)

**Default paths**
```
export GOROOT=~/go
export GOPATH=~/go
export PATH=$PATH:$GOPATH/bin
```

**Run a file**
```
go run {file}
```

**Complies a file**
```
go build {file}
./{file} -- execute the compiled file
```

**Check Golang variable**
```
go env {go_variable}
```
