Webpack
=======

**helpful links**
* [homepage](https://webpack.js.org)
* [concepts](https://webpack.js.org/concepts)
* [configuration](https://webpack.js.org/configuration)
* [CLI](https://webpack.js.org/api/cli/)

**Run with mode**
```
webpack --production
webpack --development
```

**Watch**
```
webpack --watch
```

**Run Webpack dev server**
```
webpack-dev-server [--port {port}]
```
